VISIONOPS BUSINESS OPERATIONS PLATFORM

PROJECT PURPOSE
VisionOps is a working browser-based business prototype that helps organizations monitor operational signals, prioritize incidents, review AI-assisted recommendations, estimate labor savings, and prepare management summaries.

FILES
- index.html        Business landing page and value proposition
- dashboard.html    Interactive operations center and ROI calculator
- projects.html     Product solutions, workflow, and responsible-AI safeguards
- about.html        Business purpose, creation process, and limitations
- contact.html      Accessible demonstration request form
- styles.css        Shared responsive design and dark mode
- script.js         Metrics, filters, recommendations, export, calculator, and form logic
- images/           Original project illustrations
- .vscode/          Existing local editor settings

HOW TO TEST
1. Open the project folder in Visual Studio Code.
2. Right-click index.html and choose Open with Live Server.
3. Test every navigation link.
4. Open Operations and test Refresh Data, severity filtering, row selection, Mark Action Reviewed, Export Summary, and the ROI calculator.
5. Test Dark Mode and refresh the page to confirm the preference is remembered.
6. Open Request Demo and test the required fields and confirmation message.

HOW TO PUBLISH
Upload the entire McClaney_visionops_semester_project folder contents to the same public web directory. Keep the HTML, CSS, JavaScript, and images folder together so all relative links continue to work.

RESPONSIBLE USE
This prototype uses simulated business data. Recommendations are clearly labeled as AI-assisted and require human review. It does not transmit form data or make automatic high-impact decisions.
